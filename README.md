## 📌 Описание проекта 
Создание трехмерной модели с последующим представлением

## 🚀 Что реализовано?
Реализовано моделирование модели (полностью), а также работа над окруающим миром.

## 🛠️Используемые технологии?

Использовались основные знания - Редактирование формы объекта, Работа над материалами (цветами) объекта, а также Настройка окружения и отображения (рендер-движка)


Дополнительно была использована настройка Альфа (для придания материалу Прозрачности)
